What are the different ways to select an element in the DOM?

There are many ways to select elements in the DOM. Here is a list of different ways to select an element in
the DOM : getElementById, getElementsByClassName, getElementsByTagName, querySelector, querySelectorAll, getAttribute, etc.

What is the difference between innerHTML, innerText, and textContent ?

innerHTML: Retrieves or sets the HTML content inside an element, including any HTML tags.
innerText: Retrieves or sets the visible text content, ignoring HTML tags and hidden elements.
textContent: Retrieves or sets all text inside an element, including hidden content, without parsing HTML tags.

What is event delegation in the DOM?

Event delegation is a technique where a single event listener is added to a parent element to handle events for its child elements.
This leverages event bubbling, improving performance and simplifying code, especially with dynamic content.

What is event bubbling in the DOM?

Event bubbling is a process where an event triggered on a child element propagates upward to its parent elements .
This allows parent elements to capture and handle events from their child elements.
